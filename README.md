# Projeto-labecommerce

API Compra de Produtos 

Aplicação Back-end de servidor express com banco de dados sqlite3, se trata de uma API vinculada a um banco de dados real.<br>

## Conteúdos abordados

- NodeJS
- Typescript
- Express
- SQL e SQLite
- Knex
- Postman
